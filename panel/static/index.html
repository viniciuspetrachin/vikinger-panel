Game updates

Control Valheim updates via valheim-updater (SteamCMD).

Valheim updates may break mods. Back up first. Check each mod's compatibility after updating the game.

Vanilla disables BepInEx and turns off all mods. With mods enables BepInEx and the bundled ValheimRcon.
